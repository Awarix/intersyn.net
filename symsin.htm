SymSin

Программа реконструкции схемы на основе символьного выражения схемного определителя.

Автор - Токарев Ю. В. (студент 5-го курса факультета информационных систем и технологий Ульяновского Государственного Технического Университета).

Программный продукт SymSin предназначен для восстановления электрических схем, состоящих из сопротивлений, проводимостей, индуктивностей, емкостей, а также управляемых источников, по заданному символьному выражению определителя, а также символьным выражениям передаточных и входных характеристик схемы.

Состав:
- модуль восстановления схем SymSin.exe;
- модуль графического интерфейса SymSinW.exe;
- динамическая библиотека по расчету схемных определителей CirToOut.dll;
- файл настроек SymSin.ini.

Рекомендуемые системные требования:
- ОС Windows 98/2000/Me/XP.
- 8Мб свободного места на жестком диске;
- процессор с тактовой частотой 1.8 ГГц;
- 64 Мб оперативной памяти.

Основные возможности программы:
- Восстановление схем по заданному схемному определителю;
- Восстановление схем по заданной передаточной характеристике;
- Восстановление схем по заданной входной характеристике;
- Восстановление схем по заданной схеме с эквивалентными передаточными характеристиками или схемным определителем.

Дополнительные возможности:
- Возможность исключения изоморфных схем.
